# Brand

The name plays on **refraction** — a prism splitting one beam of light into a
spectrum, mirroring Refract's job: one URL → every viewport, every angle.

## Assets
- **Mark:** `docs/icon.svg` — the prism. Theme-safe (reads on light or dark). Also the favicon source.
- **Social preview:** `docs/social-preview.png` (1280×640) — upload under GitHub repo **Settings → Social preview** so shared links render a branded card.
- **Favicon:** `docs/favicon.png` (256×256).
- Regenerate the rasters from the mark: `node scripts/gen-brand.mjs`.

## Palette
| role | hex |
|---|---|
| Accent (indigo) | `#6366f1` |
| Dark surface | `#0b1020` |
| Ink | `#f8fafc` / `#e2e8f0` |
| Muted | `#94a3b8` |
| Spectrum | `#ef4444` `#f97316` `#eab308` `#22c55e` `#3b82f6` `#8b5cf6` |

Keep it restrained: a prism mark + the indigo accent + the spectrum as a thin
accent line. Clean and technical, not a rainbow explosion — the audience is developers.
