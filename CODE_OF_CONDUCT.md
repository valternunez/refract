# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive
behavior are not tolerated. Report concerns privately to the maintainer through
GitHub (the repo's Security → "Report a vulnerability" channel also reaches the
maintainer); reports are handled confidentially.
